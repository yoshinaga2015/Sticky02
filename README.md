# Sticky

**モニターに付箋を貼るのをやめよう。**  
キーボードから手を離さずに、0.2秒で呼び出せる macOS 用メモアプリ。

## 概要

Sticky は、macOS のメニューバーに常駐し、いつでも爆速で呼び出せる超軽量メモアプリです。「**作業の流れを止めないためのツール**」として設計されています。

## 特徴

- **爆速の呼び出し**: グローバルショートカットで 0.2 秒でアクセス。画面がどんな状態でも最前面へ。
- **キーボード・ファースト**: メモのためにマウスを探す必要はありません。キーボードから手を離さずにすべてが完結します。
- **オープンソース & ローカル専用**: コードはすべて公開されており、データはローカルにのみ保存されます。パスワードや API キーを書いても安全です。
- **ミニマル設計**: 装飾や余計な機能は一切なし。書くことだけに特化しています。

## システム要件

- macOS 13.5 以降
- Apple Silicon または Intel Mac

## インストール方法

1. [リリースページ](https://github.com/yoshinaga2015/Sticky02/releases)から最新の DMG ファイルをダウンロード
2. ダウンロードした ZIP ファイルを展開し、DMG ファイルを開く
3. DMG 内の `Sticky.app` を「アプリケーション」フォルダにドラッグ&ドロップ
4. アプリケーションを起動

### 初回起動時のセキュリティ警告について

このアプリは公証されていないため、初回起動時に macOS がセキュリティ警告を表示します。

以下の手順で許可してください：

1. `Sticky.app` を右クリック → 「開く」を選択
2. または、システム環境設定の「セキュリティとプライバシー」から「このまま開く」をクリック

## 使い方

### キーボードショートカット
- **グローバル表示切替**: `Option + Space` (設定で変更可能)
- **新規メモ作成**: `Command + N`
- **リストから選択**: `Up / Down` キー
- **編集の完了**: `Command + Return`
- **メモの削除**: `Command + Delete`

### ローカル保存
Sticky に書いた内容は、外部サーバーに送信されることはありません。  
**「もう付箋にパスワードは書かないようにしましょう。」**

## 開発

### ビルド要件

- Xcode 15.0 以降
- macOS 13.5 以降

### ビルド方法

```bash
# プロジェクトを開く
open Sticky.xcodeproj

# Xcodeでビルド
# Product → Build (⌘B)
```

│   └── Repositories/    # データリポジトリ
├── StickyTests/         # ユニットテスト
└── StickyUITests/       # UIテスト
