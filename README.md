# Sticky

シンプルで美しいメモアプリ for macOS

## 概要

Stickyは、macOSのメニューバーに常駐するシンプルなメモアプリです。余計な機能は一切なく、メモを書くことに集中できるミニマルなインターフェースを提供します。

## 特徴

- **シンプルなメモ**: 余計な機能は一切なし。メモを書くことに集中できる、ミニマルなインターフェース
- **メニューバー常駐**: メニューバーに常駐するので、いつでも素早くアクセスできます
- **プライバシー重視**: データはすべてローカルに保存されます。クラウドに送信されることはありません
- **軽量**: シンプルな設計により、軽量で高速に動作します

## システム要件

- macOS 13.5 以降
- Apple Silicon または Intel Mac

## インストール方法

1. [リリースページ](https://github.com/yoshinaga2015/Sticky02/releases)から最新のDMGファイルをダウンロード
2. ダウンロードしたZIPファイルを展開し、DMGファイルを開く
3. DMG内の`Sticky.app`を「アプリケーション」フォルダにドラッグ&ドロップ
4. アプリケーションを起動

### 初回起動時のセキュリティ警告について

このアプリは公証されていないため、初回起動時にmacOSがセキュリティ警告を表示します。

以下の手順で許可してください：

1. `Sticky.app`を右クリック → 「開く」を選択
2. または、システム環境設定の「セキュリティとプライバシー」から「このまま開く」をクリック

## 使い方

1. メニューバーのStickyアイコンをクリック
2. メモを入力
3. 自動的に保存されます

## 開発

### ビルド要件

- Xcode 15.0 以降
- macOS 13.5 以降

### ビルド方法

```bash
# プロジェクトを開く
open Sticky.xcodeproj

# Xcodeでビルド
# Product → Build (⌘B)
```

### プロジェクト構造

```
Sticky/
├── Sticky/              # メインアプリケーション
│   ├── StickyApp.swift   # アプリエントリーポイント
│   ├── Views/           # SwiftUIビュー
│   ├── Models/          # データモデル
│   └── Repositories/    # データリポジトリ
├── StickyTests/         # ユニットテスト
└── StickyUITests/       # UIテスト
```
