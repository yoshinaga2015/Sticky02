Sticky

シンプルで美しいメモアプリ for macOS

ダウンロード (DMG)

macOS 13.5 以降が必要です

リリースページからDMGファイルをダウンロードしてください

シンプルなメモ

余計な機能は一切なし。メモを書くことに集中できる、ミニマルなインターフェースです。

メニューバー常駐

メニューバーに常駐するので、いつでも素早くアクセスできます。作業の流れを中断しません。

プライバシー重視

データはすべてローカルに保存されます。クラウドに送信されることはありません。

シンプルで美しい

メニューバーから素早くアクセスできる、シンプルなメモアプリ

インストール手順

  1. ダウンロード

    上記の「ダウンロード」ボタンからGitHub Releasesページに移動し、DMGファイルをダウンロードします。

  2. DMGファイルを開く

    ダウンロードしたDMGファイルをダブルクリックして開きます。

  3. アプリをインストール

    DMG内のSticky.appを「アプリケーション」フォルダにドラッグ&ドロップします。

  4. 初回起動時のセキュリティ警告について

    このアプリは公証されていないため、初回起動時にmacOSがセキュリティ警告を表示します。
    Sticky.appを右クリック → 「開く」を選択するか、システム環境設定の「セキュリティとプライバシー」から「このまま開く」をクリックしてください。

システム要件

  • macOS 13.5 以降
  • Apple Silicon または Intel Mac
