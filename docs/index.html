Sticky

モニターに付箋を貼るのをやめよう。

ダウンロード (DMG)

macOS 13.5 以降が必要です

爆速の呼び出し

0.2秒で呼び出せる。画面がどんなに散らかっていても、ショートカットひとつで最前面へ。

キーボードに集中

メモを取るためにマウスを探す必要はありません。キーボードから手を離さずにすべてが完結します。

オープンソース

中身が公開されているからこそ、安全さを保証できます。バックドアも、意図しない通信もありません。

プライバシーはあなたの手の中に

データは完全にローカル保存。クラウドに上がることはありません。

もう付箋にパスワードは書かないようにしましょう。

パスワードやAPIキーを書いても大丈夫な、あなただけの安全な場所です。

シンプルで美しい

メニューバーから素早くアクセスできる、シンプルなメモアプリ

インストール手順

  1. ダウンロード

    上記の「ダウンロード」ボタンからDMGファイルをダウンロードします。

  2. DMGファイルを開く

    ダウンロードしたDMGファイルをダブルクリックして開きます。

  3. アプリをインストール

    DMG内のSticky.appを「アプリケーション」フォルダにドラッグ&ドロップします。

  4. 初回起動時のセキュリティ警告について

    このアプリは公証されていないため、初回起動時にmacOSがセキュリティ警告を表示します。
    Sticky.appを右クリック → 「開く」を選択するか、システム環境設定の「セキュリティとプライバシー」から「このまま開く」をクリックしてください。

システム要件

  • macOS 13.5 以降
  • Apple Silicon または Intel Mac
